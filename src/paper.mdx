## Abstract

Model-based reinforcement learning agents learn a world model and improve their behavior by imagining future trajectories. But in most agents, imagination still begins from states in the replay buffer. This makes learning efficient once useful experience exists, yet slow when rewards are hidden behind rare bottlenecks.

**Mind Dreamer** untethers imagination from replay. It generates plausible latent anchors near promising frontiers, then starts imagined rollouts from those anchors so the policy can practice around bottlenecks before random exploration reaches them.

Across DeepMind Control tasks, this simple shift makes learning faster on average and especially helpful in sparse-reward settings, where standard imagination remains trapped near familiar trajectories.

## Motivation

World models are attractive because they let agents learn from imagined experience instead of expensive environment interaction. However, standard imagination is still historically tethered: rollouts usually start from states the agent has already visited.

This creates a mismatch. The world model can learn useful local structure from dense prediction losses, while the policy may still wait for sparse rewards to arrive through slow trajectory-bound exploration. In tasks with narrow gateways or delayed reward, the agent repeatedly improves what it already knows instead of practicing near the missing transition.

The question behind Mind Dreamer is therefore deliberately practical: **can a learned world model suggest useful places to start imagination, without letting the agent exploit unrealistic hallucinations?**

## How It Works

Mind Dreamer turns the world model from a passive replay machine into an active curriculum generator. It does not ask the model to invent arbitrary futures. Instead, it searches for short, plausible latent jumps that expand where the policy can practice.

In the synthetic three-ring environment, random exploration rarely finds the narrow gateways between rings. DreamerV3 therefore spends many updates improving behavior near the starting region. Mind Dreamer instead places imagined practice near candidate gateways, turning a long discovery problem into a more local refinement problem.

**Observed effect:** Mind Dreamer reaches the first critical transition **4.2x faster** by targeting high-uncertainty ring boundaries. This toy example mirrors the core intuition of the full method: use the world model to decide where imagination should begin, not only how it should unfold.

## Conclusions

Mind Dreamer shows that a world model can be more than a simulator of futures from the replay buffer. It can also propose where imagination should begin.

The central takeaway is that **better starting points for imagination can improve exploration**. Generated anchors act like a curriculum, focusing rollouts on regions that are useful, uncertain, and close enough to the learned manifold to remain trustworthy.

The approach is most helpful when discovery is hard: sparse rewards, narrow transitions, and disconnected regions where replay-tethered imagination keeps practicing what the agent already knows. The technical details, derivations, and full ablations are available in the paper and archive links above.
