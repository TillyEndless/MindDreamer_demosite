## Abstract

Model-based reinforcement learning agents learn a world model and improve their behavior by imagining future trajectories. But most agents still start imagination from historical replay states. This creates a learning asymmetry: the world model can quickly recover the latent structure of the environment, while the policy remains tied to slow sparse-reward exploration.

**Mind Dreamer** breaks this historical tether. It uses a cognitive intervention agent to search the learned latent manifold for states that are worth learning from, while manifold constraints keep generated states physically plausible.

The selected states become new anchors for imagined rollouts, letting the agent jump toward globally informative regions instead of waiting for random exploration to arrive there.

## Motivation

World models are attractive because they let agents learn from imagined experience instead of expensive environment interaction. However, standard imagination is historically tethered: rollouts usually start from states the agent has already visited.

This tether creates a learning asymmetry. The world model learns dense geometric structure from prediction losses, so it may already understand how different regions of the latent manifold connect. The policy, in contrast, only receives sparse reward signals and remains trapped in trajectory-bound exploration.

In bottleneck tasks, this means the agent repeatedly practices near familiar states while the most valuable regions for learning remain just outside the replay buffer.

Mind Dreamer asks: can the world model guide the policy toward **globally valuable learning regions** while staying on the manifold of plausible states?

## How It Works

Mind Dreamer keeps Dreamer's imagined rollouts, but changes how some rollout starts are chosen. A cognitive intervention agent proposes candidate latent states, evaluates how useful they are, and keeps only states that remain consistent with the learned manifold.

## Conclusions

Mind Dreamer shows that the bottleneck in model-based RL is not only how accurately the world model predicts, but also where imagination is allowed to begin.

By introducing a cognitive intervention agent, Mind Dreamer turns the learning asymmetry into an advantage: the world model's fast geometric learning is used to locate state regions with high expected learning value. Relay Expected Free Energy scores these regions, manifold constraints keep them grounded, and contrastive training helps select anchors that are more valuable than historical replay states.

The result is a targeted form of exploration: instead of diffusing slowly through history, the agent can jump toward globally worthwhile regions and learn from them. The technical details, derivations, and full ablations are available in the paper and archive links above.
